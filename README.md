

## Supply Chain Data Analysis & Visuallization
## Overview
This project focuses on analyzing and visualizing Supply chain data to identify Performance.

The dataset used in this project was Sourced from Kaggle.

## Dataset
Supply chain Dataset (Kaggle)
The dataset includes information such 
as
-Orders
-Shipping details
-Delivery Performance

## Tools & Technologies Used 
-Python
-Pandas
-Matplotlib
-Seaborn
-Jupyter Notebook

## Project Objectives 
-Perform data cleaning and preprocessing
-Coduct exploratory data analysis
-Visualize key supply chain metrics
-Identify patterns in shipping and delivery performance

## Visualizations Created
-Bar Charts
-Pie Charts
-Histograms

## How to Run
1. Download The dataset from Kaggle
2. Open the vs code Jupyter Notebook (.ipynb file)
3. Run all cells
